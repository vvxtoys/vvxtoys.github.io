使用nexus3.x搭建maven私服

发表于 2018-09-19 | 分类于 Java | | 阅读次数

前言

好久之前就想搭建maven仓库了，一直拖到了现在，也就是懒得动，现在终于是要付诸行动了。网上查了不少资料，好多博客都是关于2.x的搭建，我下载的是最新版的nexus，好多教程已经不能使用，以此记录我的踩坑搭建之路。

安装环境和需要的软件

  • Centos 7
  • Java 1.8
  • nexus OSS 3.13

Java安装

  • 卸载openjdk
  • 安装

Nexus安装

  • 创建文件夹
  • 下载安装文件
  • 解压

解压后

  • 修改配置文件 也可以不改

默认端口8081,我修改为5743

  • 开放端口 root用户

  • 启动nexus

  • 本地访问 ip:端口号 默认账户admin 密码 admin123

  • 选择Repositories 创建三方私有库

  • 我创建了两个maven hosted仓库

  • 将这两个仓库加入到maven-public中

    点击maven-public

    到这里线上就配置完成了

配置Maven

  • 找到setting.xml 在mirrors中添加mirror
  • 在servers中添加server
  • 在profiles中添加profile
  • 激活profile

上传jar 发布项目

  • 网页上传jar upload

  • maven deploy

    • pom.xml 添加 distributionManagement

    id要和setting.xml中的server id一致

    • 运行maven deploy

  • 查看线上


发布成功,以后可以直接在别的项目中使用
